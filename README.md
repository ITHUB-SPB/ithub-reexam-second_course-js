# Пересдача дисциплины "JavaScript" за 2 курс I семестр

## Установка и настройка

Клонируйте репозиторий через `git clone https://github.com/ITHUB-SPB/ithub-reexam-second_course-js.git`

Откройте директорию в `VS Code` (File -> Open Folder)

Установите зависимости через `npm i` в консоли.

## Порядок работы

Для запуска тестов используйте расширение `Vitest` для `VS Code` либо команду `npm run test` в консоли.

Для проверки на ошибки линтером используйте `npm run lint`.

Для форматирования используйте `npm run format`.

## Критерии оценивания

Максимальный балл по критериям - **14**, который далее будет умножен на 5 (чтобы попасть в шкалу до 0 до 70 баллов).

Таким образом, **минимальный проходной балл - это 10**.

Чтобы отслеживать, насколько вы близки к победе, время от времени используйте команду `npm run test:report`.
