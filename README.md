# Пересдача дисциплины "JavaScript" за 2 курс I семестр

## Установка и настройка

Клонируйте репозиторий через `git clone https://github.com/ITHUB-SPB/ithub-reexam-second_course-js.git`

Откройте директорию в `VS Code` (File -> Open Folder)

Установите зависимости через `npm i` в консоли.

Для запуска тестов можно использовать расширение `Vitest` для `VS Code` либо выполнить команду `npm run test` в консоли.

Для подсчета итогового балла за тесты используйте `npm run test:report`.

Для проверки на ошибки линтером используйте `npm run lint`.

Для форматирования используйте `npm run format`.

## Критерии оценивания

_Максимальный балл по критериям - 10. Итоговый балл за пересдачу будет скорректирован множителем, который зависит от максимального доступного вам балла за пересдачу._

#### Основные (0 - 9 баллов)

- до девяти баллов можно получить за прохождение автотестов

#### Бонусы (0 - 1 балла)

- код аккуратно отформатирован (встроенными средствами, приттиером или аналогом, либо вручную)
- код не содержит ошибок линтера
- работа сдана путём загрузки в личный репозиторий гитхаба (либо аналога)
